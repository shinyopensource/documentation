CATrend Analyzer

A Tool for Performing the Cochran-Armitage Test for Trend

R-Shiny MIT License Free to Use
▶ Launch Live App View on GitHub ⬇ Example Dataset

Overview

CATrend Analyzer is a free, web-based R-Shiny application designed specifically for detecting trends in proportions across ordered categorical variables. This test is widely used in clinical trials, epidemiology, and public health research, particularly when assessing whether the probability of a binary outcome changes consistently across increasing levels of an exposure or dose. The tool eliminates the need for manual calculations, requires no software installation or programming skills, and is completely free to use.

Complete Analysis Workflow

Follow these steps inside the CATrend Analyzer application to perform a complete trend analysis.

1
Select Data Type

Choose either "Normal Data" (individual-level dataset) or "Crosstabulation Data" (pre-summarized count matrix).

2
Upload Your Data File (Normal Data Mode)

Select file type (CSV, Excel, Stata, SPSS) and browse to upload your data. For CSV files, customize delimiter selection.

3
Setup Crosstabulation Table (Crosstabulation Mode)

Specify names for dependent and independent variables, define levels, and enter counts in an interactive table.

4
Review Missing Data & Data Quality

Check missing data summaries and use the "Remove Rows with Missing Values" button to clean the data.

5
Select Variables

Choose the dependent variable (binary outcome) and the independent variable (ordered exposure or grouping variable).

6
Order Independent Variable Levels

Use drag-and-drop bucket lists to arrange levels from lowest to highest (e.g., Placebo → Low → Medium → High).

7
Run the Cochran-Armitage Test

Click "Run Cochran-Armitage Test" to generate Z-statistic, chi-square statistic, p-value, and intermediate calculations.

8
Customize Visualization

Create customizable stacked bar plots displaying counts, row percentages, or column percentages with custom colors.

9
Download Full Report

Click "Download Full Report" to obtain a Word document containing the complete analysis, graph, and results.

Statistical Formula

The Cochran-Armitage test for trend evaluates whether there is a linear trend in proportions across ordered groups. The test statistic is calculated as:

Z = [Σ yj (xj - x̄)] / √[p̄(1-p̄) Σ nj (xj - x̄)²]

Where:

  • xj = scores assigned to ordered groups (1, 2, 3, ..., k)
  • yj = number of "successes" (events) in group j
  • nj = total number of observations in group j
  • x̄ = weighted mean of scores = Σ(njxj) / Σnj
  • p̄ = overall proportion of successes = Σyj / Σnj
📐 Continuity Correction

For small samples, the app can apply a continuity correction of c = 0.5 × (score spacing) to improve approximation accuracy.

Test Interpretation

Z Positive (+Z)

Increasing trend — As the ordered groups increase (e.g., dose, time), the proportion of the event (first level of the outcome variable) increases.

Z Negative (-Z)

Decreasing trend — As the ordered groups increase, the proportion of the event decreases.

📊 p-value Interpretation

p-value < 0.05: Statistically significant trend at α = 0.05. The test statistic Z follows a standard normal distribution; the chi-square equivalent is Z² with 1 degree of freedom.

Assumptions of the Cochran-Armitage Trend Test

🔗
Independence

Observations are independent of each other.

📊
Ordered Categories

Independent variable levels have a natural order.

📈
Monotonic Trend

The trend is assumed to be monotonic (consistently increasing or decreasing).

🔢
Adequate Sample Size

Expected cell counts should ideally be >5 for reliable results.

⚖️
Binary Outcome

The test is designed for binary dependent variables.

Limitations

The following known limitations apply to the current version of CATrend Analyzer:

📉
Small Sample Sizes

Results may be unreliable when expected cell counts are less than 5.

🔄
Non-monotonic Trends

The test may not detect non-linear or non-monotonic patterns.

⚠️
No Confounder Adjustment

The test does not adjust for other variables or confounders.

🧮
Sparse Data

Many zero cells can affect test accuracy.
