MedModr

An Open-Source Mediation & Moderation Software for Structural Equation Modeling

MedModr is a free, web-based R-Shiny application that provides an intuitive interface for conducting mediation and moderation analyses. Built on the powerful lavaan package, it enables researchers to test complex statistical models without requiring advanced programming skills.

Before You Begin:
  • Ensure your data is clean with no missing values (or be aware that rows with missing values will be removed)
  • Variable names should not contain spaces or special characters
  • For categorical variables, consider creating dummy variables beforehand
  • Have at least 10-20 observations per variable for reliable estimates

Prerequisites

  • R version 4.4 or higher (for local installation)
  • RStudio (recommended for local development)
  • A modern web browser (for web version)

Steps to Run Locally

1. Download app.R from GitHub
Access the source code at: https://github.com/mudassiribrahim12/MedModr/blob/main/app.R

2. Install required R packages

# Install required packages
install.packages(c(
  "shiny", "lavaan", "semPlot", "ggplot2", "readr", "readxl",
  "haven", "DT", "colourpicker", "officer", "magick", "flextable",
  "shinythemes", "shinyjs", "shinydashboard"
))

3. Load required packages

4. Run the application in RStudio

Simple Mediation Analysis - Step-by-Step Guide

Test whether the effect of X on Y is transmitted through a mediator M (X → M → Y).

1
Select Your Interface
  • On startup, choose "Interactive Interface" for point-and-click selection
  • Or choose "Syntax Interface" if you prefer writing lavaan code
  • For beginners, the Interactive Interface is recommended
2
Load Your Data
  • Click "Browse" in the Data Input section
  • Select your data file (CSV, XLSX, Stata or SPSS)
  • Wait for the data preview to appear in the "Variables in Data" tab
  • Verify that all your variables are correctly loaded and displayed
3
Select Analysis Type
  • From the dropdown menu, select "Simple Mediation"
  • Three variable selection boxes will appear: X, M, and Y
4
Select Your Variables
  • Click on the X box (Independent Variable) and select your predictor variable
  • Click on the M box (Mediator Variable) and select your mediator variable
  • Click on the Y box (Dependent Variable) and select your outcome variable
  • Example: X = "Stress", M = "Anxiety", Y = "Performance"
Tip: Variables are automatically validated — each box will display the selected variable name.
5
Run the Analysis
  • Click the "Run Analysis" button (blue button with play icon)
  • Wait for the analysis to complete (you'll see a notification when done)
  • The Results tab will automatically open with your output
6
Interpret Your Results
  • Path Estimates: Look at a-path (X→M) and b-path (M→Y) significance (p < 0.05)
  • Indirect Effect: Check if a*b is significant - this indicates mediation
  • Direct Effect (c'): If not significant with significant indirect effect = full mediation
  • Total Effect (c): Overall effect of X on Y
Interpretation Example:
"The indirect effect of Stress on Performance through Anxiety was significant (β = -0.15, p < .001), indicating that Anxiety mediates this relationship."
7
Visualize & Export
  • Go to "Diagram" tab to view and customize your path diagram
  • Adjust layout, colors, and node labels as desired
  • Click "Download Diagram (PNG)" to save as PNG
  • Click "Download Word Report" to generate a complete analysis report

Serial Mediation Analysis - Step-by-Step Guide

Test sequential pathways where mediators are ordered (X → M1 → M2 → Y).

1
Load Your Data

Same as Simple Mediation - upload your data file and verify variables load correctly.

2
Select Analysis Type

From the dropdown, select "Serial Mediation". Four variable boxes will appear: X, M1, M2, and Y.

3
Select Variables in Order
  • X (Independent Variable): Your predictor (e.g., "Workload")
  • M1 (First Mediator): The first mechanism (e.g., "Stress")
  • M2 (Second Mediator): The second mechanism (e.g., "Burnout")
  • Y (Dependent Variable): Your outcome (e.g., "Job Satisfaction")
Important: M1 and M2 must be in theoretical sequential order.
4
Run the Analysis
  • Click the "Run Analysis" button (blue button with play icon)
  • Wait for the analysis to complete (you'll see a notification when done)
  • The Results tab will automatically open with your output
5
Interpret Results
  • Indirect Effect M1: X → M1 → Y pathway (a1 × b1)
  • Indirect Effect M2: X → M2 → Y pathway (d × b2)
  • Serial Indirect Effect: X → M1 → M2 → Y pathway (a1 × a2 × b2)
  • Total Indirect Effect: Sum of all indirect pathways
  • Total Effect: c + Total Indirect Effect
Interpretation Example:
"The serial indirect effect (Workload → Stress → Burnout → Satisfaction) was significant (β = -0.08, p < .01), supporting the hypothesized sequential mediation process."
6
Visualize & Export
  • Go to "Diagram" tab to view and customize your path diagram
  • Adjust layout, colors, and node labels as desired
  • Click "Download Diagram (PNG)" to save as PNG
  • Click "Download Word Report" to generate a complete analysis report

Moderation Analysis - Step-by-Step Guide

Test whether the effect of X on Y depends on the level of moderator W.

1
Load Your Data

Upload your data file with X (independent), W (moderator), and Y (dependent) variables.

2
Select Analysis Type

From dropdown, select "Moderation". Three variable boxes appear: X, W, and Y.

3
Select Variables
  • X (Independent Variable): Your predictor (e.g., "Study Time")
  • W (Moderator Variable): The condition that modifies the relationship (e.g., "Sleep Quality")
  • Y (Dependent Variable): Your outcome (e.g., "Exam Score")
Automatic: The app automatically creates centered variables and the interaction term (X_W) when you run the analysis!
4
Run the Analysis
  • Click the "Run Analysis" button (blue button with play icon)
  • Wait for the analysis to complete (you'll see a notification when done)
  • The Results tab will automatically open with your output
5
Interpret Results
  • Interaction Term (X_W): If significant (p < 0.05), moderation is present
  • Simple Slopes: Check the effect of X on Y at different W levels:
    • Low W (-1 SD): slope = b1 + b3 × (-1)
    • Average W (mean): slope = b1
    • High W (+1 SD): slope = b1 + b3 × (+1)
  • Direction: Positive b3 = effect strengthens as W increases; Negative b3 = effect weakens
Interpretation Example:
"The interaction between Study Time and Sleep Quality was significant (β = 0.15, p = .002). Simple slopes analysis revealed that Study Time positively predicted Exam Score only when Sleep Quality was high (+1 SD: β = 0.32, p < .001), but not when Sleep Quality was low (-1 SD: β = 0.02, p = .85)."
6
Visualize with Slope Plot
  • Go to "Diagram" tab and select "Slope Plot"
  • View the conditional effect at different moderator levels
  • Customize colors, labels, and download the plot
  • The Simple Slopes Values panel shows exact coefficients
  • Click "Download Slope Plot (PNG)" to save as PNG
  • Click "Download Word Report" to generate a complete analysis report

    • Moderated Mediation (Conditional Process) - Step-by-Step Guide

    Test whether the indirect effect of X on Y through M depends on moderator W (path a moderation).

    1
    Load Your Data

    Upload data with X (independent), M (mediator), W (moderator), and Y (dependent).

    2
    Select Analysis Type

    From dropdown, select "Moderated Mediation". Four variable boxes appear: X, M, W, and Y.

    3
    Select Variables
    • X (Independent Variable): Your predictor (e.g., "Leadership Support")
    • M (Mediator Variable): The mechanism (e.g., "Job Engagement")
    • W (Moderator Variable): The condition (e.g., "Organizational Climate")
    • Y (Dependent Variable): Your outcome (e.g., "Performance")
    Note: The app automatically creates the X×W interaction term and centers variables for the path a equation.
    4
    Run the Analysis
    • Click the "Run Analysis" button (blue button with play icon)
    • Wait for the analysis to complete (you'll see a notification when done)
    • The Results tab will automatically open with your output
    5
    Interpret Results
    • Path a (X → M) Moderated by W: Check a3 coefficient (X×W → M) - if significant, the a-path varies across W levels
    • Path b (M → Y): Should be significant for mediation to occur
    • Conditional Indirect Effects:
      • Low W (-1 SD): (a1 + a3×-1) × b
      • Average W (mean): a1 × b
      • High W (+1 SD): (a1 + a3×+1) × b
    • Index of Moderated Mediation (a3 × b): If significant (p < 0.05), the indirect effect significantly differs across W levels - this is evidence of moderated mediation
    Interpretation Example:
    "The index of moderated mediation was significant (β = 0.08, p = .01), indicating that the indirect effect of Leadership Support on Performance through Job Engagement varied significantly across levels of Organizational Climate. The conditional indirect effect was significant only when Organizational Climate was high (β = 0.24, p < .001) but not when low (β = 0.05, p = .32)."
    6
    Export
    • Click "Download Word Report" to generate a complete analysis report

    Limitations

    • Latent Variables: Current version focuses on observed variable models
    • Missing Data: Listwise deletion only (cases with missing values are removed)
    • Large Datasets: >10,000 rows are truncated for performance
    • Bootstrap: Limited to 5,000 samples
