MedModr

An Open-Source Mediation & Moderation Software

MedModr is a free, web-based R-Shiny application that provides an intuitive interface for conducting mediation and moderation analyses. Built on the powerful lavaan package, it enables researchers to test complex statistical models without requiring advanced programming skills.

Before You Begin:
  • Ensure your data is clean with no missing values
  • Variable names should not contain spaces or special characters
  • Have at least 10-20 observations per variable for reliable estimates

Prerequisites

  • R version 4.4 or higher
  • RStudio (recommended)

Steps to Run Locally

1. Download app.R from GitHub
Access the source code at: https://github.com/mudassiribrahim12/MedModr/blob/main/app.R

2. Install required R packages

install.packages(c(
  "shiny",
  "lavaan",
  "semPlot",
  "ggplot2",
  "readr",
  "readxl",
  "haven",
  "DT",
  "colourpicker",
  "officer",
  "magick",
  "flextable",
  "shinythemes",
  "shinyjs",
  "shinydashboard"
))

3. Load required R packages

# Load libraries
library(shiny)
library(shinydashboard)
library(DT)
library(haven)
library(readxl)
library(ggplot2)
library(officer)
library(rmarkdown)
library(tidyverse)
library(sortable)
library(scales)

4. Run the application in RStudio

# Set your working directory to where you saved app.R
setwd("C:/Users/YourName/Desktop/")    # Windows example
# or
setwd("/home/yourname/Desktop/")       # Linux/Mac example

# Load the shiny library
library(shiny)

# Run the application
runApp("app.R")
Replace the path with the actual location where you saved the app.R file.

Deployment

# Deploy to shinyapps.io
install.packages("rsconnect")
rsconnect::deployApp("path/to/MedModr")

Simple Mediation Analysis - Step-by-Step Guide

Test whether the effect of X on Y is transmitted through a mediator M (X → M → Y).

1
Select Your Interface
Choose "Interactive Interface" for point-and-click selection or "Syntax Interface" for lavaan code.
2
Load Your Data
Click "Browse" and select your data file (CSV, XLSX, Stata, or SPSS).
3
Select Analysis Type
From dropdown, select "Simple Mediation". Three variable boxes appear: X, M, and Y.
4
Select Your Variables
Select predictor (X), mediator (M), and outcome (Y) from the dropdown lists.
5
Run the Analysis
Click "Run Analysis" and wait for results to appear.
6
Interpret Results
Check a-path (X→M), b-path (M→Y), indirect effect (a×b), direct effect (c'), and total effect (c).
7
Visualize & Export
View path diagram, customize colors, download PNG, or generate Word report.

Serial Mediation Analysis - Step-by-Step Guide

Test sequential pathways where mediators are ordered (X → M1 → M2 → Y).

1
Load Your Data
Upload your data file with all variables.
2
Select Analysis Type
From dropdown, select "Serial Mediation". Four variable boxes appear: X, M1, M2, and Y.
3
Select Variables in Order
Select X (predictor), M1 (first mediator), M2 (second mediator), and Y (outcome) in theoretical sequence.
4
Run the Analysis
Click "Run Analysis" and wait for results.
5
Interpret Results
Review indirect effects: X→M1→Y, X→M2→Y, and serial X→M1→M2→Y pathways.
6
Visualize & Export
View path diagram and download report.

Moderation Analysis - Step-by-Step Guide

Test whether the effect of X on Y depends on the level of moderator W.

1
Load Your Data
Upload data with X, W, and Y variables.
2
Select Analysis Type
From dropdown, select "Moderation". Three variable boxes appear: X, W, and Y.
3
Select Variables
Select predictor (X), moderator (W), and outcome (Y).
4
Run the Analysis
Click "Run Analysis" - app automatically creates centered variables and interaction term.
5
Interpret Results
Check interaction term (X×W) significance and simple slopes at low (-1 SD), average, and high (+1 SD) moderator levels.
6
Visualize with Slope Plot
Go to "Diagram" tab, select "Slope Plot", customize, and download.

Moderated Mediation - Step-by-Step Guide

Test whether the indirect effect of X on Y through M depends on moderator W.

1
Load Your Data
Upload data with X, M, W, and Y variables.
2
Select Analysis Type
From dropdown, select "Moderated Mediation". Four variable boxes appear: X, M, W, and Y.
3
Select Variables
Select predictor (X), mediator (M), moderator (W), and outcome (Y).
4
Run the Analysis
Click "Run Analysis" - app creates X×W interaction term.
5
Interpret Results
Check a3 coefficient (X×W→M) significance and index of moderated mediation (a3×b).
6
Export
Download Word report with complete results.

Statistical Engine

MedModr uses the lavaan package for structural equation modeling, implementing:

  • Mediation: Product-of-coefficients method
  • Moderation: Linear regression with mean-centering and simple slopes analysis
  • Moderated Mediation: Conditional process modeling with index of moderated mediation

Path Diagrams

Customizable path diagrams using semPlot with options for layout, edge colors, node colors, and label sizes.

Simple Slopes Plot

Visualize moderation effects with customizable colors, line types, and confidence bands.

Limitations

  • Current version focuses on observed variable models
  • Listwise deletion only for missing values
  • Datasets >10,000 rows are truncated for performance
