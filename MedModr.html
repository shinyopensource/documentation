Overview

MedModr is a free, web-based R-Shiny application that provides an intuitive interface for conducting mediation and moderation analyses. Built on the powerful lavaan package for structural equation modeling, it enables researchers to test complex statistical models without requiring advanced programming skills. The tool eliminates the need for manual calculations, requires no software installation, and is completely free to use.

Common Research Problems Solved

🔗
Mediation TestingAutomatically tests whether the effect of X on Y is transmitted through a mediator M.
⚖️
Moderation AnalysisDetermines whether the effect of X on Y depends on the level of moderator W.
🔄
Serial MediationTests sequential pathways with multiple mediators in a causal chain.
📊
Conditional ProcessEvaluates whether indirect effects vary across levels of a moderator.

Local Installation

Step 1 — Download app.R

Download the source file from the GitHub repository: github.com/mudassiribrahim12/MedModr

Step 2 — Install required packages

Run the following in your R console:

install.packages(c(
    "shiny", "shinydashboard", "DT",
    "haven", "readxl", "readr",
    "lavaan", "semPlot", "ggplot2",
    "officer", "colourpicker", "magick",
    "flextable", "shinythemes", "shinyjs"
  ))

Step 3 — Load the libraries

library(shiny)
library(shinydashboard)
library(DT)
library(haven)
library(readxl)
library(readr)
library(lavaan)
library(semPlot)
library(ggplot2)
library(officer)
library(colourpicker)
library(magick)
library(flextable)
library(shinythemes)
library(shinyjs)

Step 4 — Set working directory and run

Replace the path below with the location of your saved app.R file:

# Windows
setwd("C:/Users/YourName/Desktop/")

# Linux / macOS
setwd("/home/yourname/Desktop/")

# Launch the app
library(shiny)
runApp("app.R")

Step 5 — Open in browser

The app opens in the RStudio viewer pane. Click Open in Browser for the best experience.

Deployment

Deploy your own instance to shinyapps.io or any compatible hosting platform using the rsconnect package.

# Install rsconnect
install.packages("rsconnect")

# Deploy — replace with actual path to your app folder
rsconnect::deployApp("path/to/MedModr")
⚠️ Note

Replace "path/to/MedModr" with the actual folder path containing your app.R file.

Simple Mediation Analysis - Step-by-Step Guide

Test whether the effect of X on Y is transmitted through a mediator M (X → M → Y).

1
Select Your Interface

Choose "Interactive Interface" for point-and-click selection or "Syntax Interface" for lavaan code.

2
Load Your Data

Click "Browse" and select your data file (CSV, XLSX, Stata, or SPSS).

3
Select Analysis Type

From dropdown, select "Simple Mediation". Three variable boxes appear: X, M, and Y.

4
Select Your Variables

Select predictor (X), mediator (M), and outcome (Y) from the dropdown lists.

5
Run the Analysis

Click "Run Analysis" and wait for results to appear.

6
Interpret Results

Check a-path (X→M), b-path (M→Y), indirect effect (a×b), direct effect (c'), and total effect (c).

7
Visualize & Export

View path diagram, customize colors, download PNG, or generate Word report.

Serial Mediation Analysis - Step-by-Step Guide

Test sequential pathways where mediators are ordered (X → M1 → M2 → Y).

1
Load Your Data

Upload your data file with all variables.

2
Select Analysis Type

From dropdown, select "Serial Mediation". Four variable boxes appear: X, M1, M2, and Y.

3
Select Variables in Order

Select X (predictor), M1 (first mediator), M2 (second mediator), and Y (outcome) in theoretical sequence.

4
Run the Analysis

Click "Run Analysis" and wait for results.

5
Interpret Results

Review indirect effects: X→M1→Y, X→M2→Y, and serial X→M1→M2→Y pathways.

6
Visualize & Export

View path diagram and download report.

Moderation Analysis - Step-by-Step Guide

Test whether the effect of X on Y depends on the level of moderator W.

1
Load Your Data

Upload data with X, W, and Y variables.

2
Select Analysis Type

From dropdown, select "Moderation". Three variable boxes appear: X, W, and Y.

3
Select Variables

Select predictor (X), moderator (W), and outcome (Y).

4
Run the Analysis

Click "Run Analysis" — app automatically creates centered variables and interaction term.

5
Interpret Results

Check interaction term (X×W) significance and simple slopes at low (-1 SD), average, and high (+1 SD) moderator levels.

6
Visualize with Slope Plot

Go to "Diagram" tab, select "Slope Plot", customize, and download, and download report.

Moderated Mediation (Conditional Process) - Step-by-Step Guide

Test whether the indirect effect of X on Y through M depends on moderator W.

1
Load Your Data

Upload data with X, M, W, and Y variables.

2
Select Analysis Type

From dropdown, select "Moderated Mediation". Four variable boxes appear: X, M, W, and Y.

3
Select Variables

Select predictor (X), mediator (M), moderator (W), and outcome (Y).

4
Run the Analysis

Click "Run Analysis" — app creates X×W interaction term.

5
Interpret Results

Check a3 coefficient (X×W→M) significance and index of moderated mediation (a3×b).

6
Export

Download Word report with complete results.

Path Diagrams

MedModr generates customizable path diagrams using the semPlot package. Users can adjust layout, edge colors, node colors, label sizes, and other visual elements to create publication-ready figures.

🎨 Customization Options

Choose from multiple layout algorithms (tree, circle, spring), modify edge colors to highlight significant paths, and adjust label sizes for clarity. Diagrams can be downloaded as PNG files for inclusion in manuscripts.

Simple Slopes Plots

Visualize moderation effects with customizable simple slopes plots showing the relationship between X and Y at different levels of the moderator W (typically at -1 SD, mean, and +1 SD).

📊 Plot Features

Customize colors, line types, and confidence bands. Plots can be downloaded in PNG format for direct use in presentations and publications.

Statistical Engine

MedModr uses the lavaan package for structural equation modeling, implementing:

Analysis TypeMethodImplementation
Simple & Serial MediationProduct-of-coefficientsPath analysis
ModerationLinear regressionMean-centering, interaction term, simple slopes at ±1 SD
Moderated MediationConditional process modelingIndex of moderated mediation (a3×b)

Limitations

The following known limitations apply to the current version of MedModr:

📊
Observed Variables Only

Current version focuses on observed variable models (no latent variables).

❓
Missing Data Handling

Uses listwise deletion only. Users should pre-process missing data before uploading.

📏
Dataset Size Limit

Datasets >10,000 rows are truncated for performance. Contact developer for full-scale analysis assistance.

🌐
Internet Required for Web Version

The hosted version requires an active internet connection. For offline use, install and run the app locally.
