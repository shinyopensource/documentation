A Comprehensive Sample Size Calculation, Power Analysis & Descriptive Statistics Suite

R-Shiny MIT License Free to Use
▶ Launch Live App View on GitHub

Overview

CalcuStats is a free, web-based R-Shiny application that provides researchers, healthcare professionals, and students with reliable, validated methods for sample size determination, power analysis, and descriptive statistics. The tool supports multiple sample size formulas, stratified proportional allocation with flowchart generation, downloadable Word reports.

Problems CalcuStats Solves

📐
Sample Size DeterminationCalculates sample sizes for diverse study designs — from simple proportions to cohort and case-control studies.
⚡
Power AnalysisDetermines required sample size across t-tests, ANOVA, correlation, regression, and chi-squared tests.
🔄
Proportional AllocationDistributes sample sizes across strata with non-response adjustment and step-by-step calculations.
📄
Professional ReportingGenerates downloadable Word documents and stratification flowcharts in PNG, SVG, and PDF.

Analysis Workflow

Follow these steps inside the CalcuStats application to perform a complete sample size calculation with stratified allocation.

1
Select a Module

Choose from the top navigation: Proportional Allocation, Taro Yamane, Cochran Formula, Other Formulas, Power Analysis, or Descriptive Statistics.

2
Enter Parameters

Fill in the required inputs for your selected formula — population size, margin of error, proportion, confidence level, etc. Proportion inputs accept decimal (0.5), percentage without symbol (50), or percentage with symbol (50%).

3
Set Non-Response Rate

Enter the expected non-response rate (%). The app automatically adjusts the final sample size using: n_adjusted = n / (1 − non-response rate).

4
Add Strata (Optional)

Click Add Stratum to define sub-populations. Enter each stratum's name and population size. The app allocates the total sample proportionally across all strata.

5
Calculate

The results panel updates automatically showing the required sample size, adjusted sample size, step-by-step formula derivation, and the proportional allocation table.

6
Generate Flowchart (Optional)

Enable Generate Flow Chart Diagram to visualize the stratification. Customize node shapes, colors, fonts, and labels using the interactive editor.

7
Download Results

Click Download as Word for a complete Word document report, or Download Calculation Steps for a plain-text derivation file. Flowcharts export as PNG, SVG, or PDF.

Statistical Formulas

CalcuStats implements the following validated sample size formulas. All formulas include step-by-step derivation output in the application.

Proportional Allocation

ni = (Ni / N) × n
  • ni = sample size for stratum i
  • Ni = population of stratum i
  • N = total population size
  • n = total (adjusted) sample size

Taro Yamane Formula

n = N / (1 + N × e²)

Assumes 95% confidence level. Suitable for finite populations. Non-response adjustment applied after: n_adj = ⌈n / (1 − r)⌉.

Cochran's Formula

n₀ = (Z² × p × (1−p)) / e²
Finite correction: n = n₀ / (1 + (n₀ − 1) / N)

Use when working with proportions. Z = 1.96 for 95% CI. Optional finite population correction when N is known.

Other Formulas

Single Population Proportion (EPI INFO)

n = Z² × p × (1−p) / d²

For estimating a proportion with specified precision. Accepts proportion or percentage input.

Mean Estimation (Known Variance)

n = (Z² × σ²) / d²

When population standard deviation σ is known from prior studies.

Case-Control Study

n = [Zα√(2p̄q̄) + Zβ√(p₁q₁ + p₂q₂)]² / (p₁−p₂)²

Where p₂ = p₁ × OR / (1 + p₁(OR−1)). Supports case:control ratio.

Cohort Study

n = [Zα√(2p̄q̄) + Zβ√(p₁q₁ + p₂q₂)]² / (p₁−p₂)²

Where p₂ = p₁ × RR. Supports exposed:unexposed ratio.

Correlation Coefficient

n = [(Zα + Zβ) / (0.5 × ln((1+r)/(1−r)))]² + 3

For testing significance of a Pearson correlation coefficient.

Prevalence Study

n = (Z² × p × (1−p)) / d²

For estimating disease prevalence with specified margin of error.

📊 Recommended Power

A power of 0.80 (80%) is generally considered acceptable in most research contexts. This means an 80% probability of correctly detecting a true effect at the specified significance level.

Flowchart Generation

All sample size modules (Proportional Allocation, Taro Yamane, Cochran, Other Formulas) support optional stratification flowchart diagrams generated using DiagrammeR.

1
Enable Flowchart

Check "Generate Flow Chart Diagram" after adding 2 or more strata.

2
Customize Appearance

Choose layout (dot, neato, twopi, circo, fdp), node shape, fill color, edge color, font sizes, node dimensions, and arrow size.

3
Customize Content

Toggle whether to display population size and stratum sample size inside each node. Edit node and edge labels interactively by clicking on diagram elements.

4
Download Diagram

Export as PNG (3000×2000px, high-quality raster), SVG (infinite-resolution vector), or PDF (print-ready vector).

Input Formats & Flexible Proportion Entry

CalcuStats accepts proportion inputs in three equivalent formats across all relevant formula inputs:

⚠️ Boundary Handling

Proportions must be between 0 and 1 (exclusive). Values entered as integers between 1 and 100 are automatically interpreted as percentages. Proportions outside valid ranges are clamped to 0.01–0.99 where applicable (e.g., case-control and cohort study formulas).

Limitations

📉
Non-Monotonic Power Curves

Power calculations assume standard two-sided tests. One-sided tests require manual adjustment of the alpha parameter.

⚠️
No Confounder Adjustment

Sample size formulas do not adjust for covariates or confounders. Multivariate designs may require additional inflation.

🔄
Session State Not Persistent

The hosted Shiny app does not save state between sessions. Download results before closing the browser.

📊
Descriptive Statistics

The descriptive statistics module is designed for single-variable (univariate) analysis. Multi-column datasets should be analyzed one variable at a time.

🌐
Lite Version Scope

The HTML Lite version covers only four calculators (Proportional Allocation, Taro Yamane, Cochran, and basic Other Formulas). Power analysis, descriptive statistics, and flowcharts require the full Shiny app.
