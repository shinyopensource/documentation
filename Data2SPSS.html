Data2SPSS

An interactive R-Shiny web application for converting data files to SPSS and other formats.

R-Shiny MIT License Free to Use
▶ Launch Live App View on GitHub ⬇ Example Dataset

Overview

Data2SPSS is a free, web-based R-Shiny application that automates the conversion of various data formats into SPSS-ready files and enables bidirectional conversion between multiple statistical software formats. The tool eliminates the need for manual data preparation, requires no software installation or programming skills, and is completely free to use.

Common Research Problems Solved

🏷️
Lost Value LabelsValue labels dropped during file transfer are automatically restored.
🔤
Special Characters in NamesVariable names with spaces or symbols are cleaned to meet SPSS requirements.
📊
Wrong Variable TypesCategorical variables not recognized correctly are automatically detected and handled.
🌐

Local Installation

Step 1 — Download app.R

Download the source file from the GitHub repository: github.com/mudassiribrahim12/Data2SPSS

Step 2 — Install required packages

Run the following in your R console:

install.packages(c(
  "shiny", "shinythemes", "readxl",
  "readr",  "haven",        "foreign",
  "rio",   "DT",          "dplyr",
  "labelled", "purrr",    "shinyjs",
  "writexl"
))

Step 3 — Load the libraries

library(shiny)
library(shinythemes)
library(readxl)
library(readr)
library(haven)
library(foreign)
library(rio)
library(DT)
library(dplyr)
library(labelled)
library(purrr)
library(shinyjs)
library(writexl)

Step 4 — Set working directory and run

Replace the path below with the location of your saved app.R file:

# Windows
setwd("C:/Users/YourName/Desktop/")

# Linux / macOS
setwd("/home/yourname/Desktop/")

# Launch the app
library(shiny)
runApp("app.R")

Step 5 — Open in browser

The app opens in the RStudio viewer pane. Click Open in Browser for the best experience.

Deployment

Deploy your own instance to shinyapps.io or any compatible hosting platform using the rsconnect package.

# Install rsconnect
install.packages("rsconnect")

# Deploy — replace with actual path to your app folder
rsconnect::deployApp("path/to/Data2SPSS")
⚠️ Note

Replace "path/to/Data2SPSS" with the actual folder path containing your app.R file, e.g. "C:/Users/YourName/Desktop/Data2SPSS".

Basic Workflow

Follow these steps inside the Data2SPSS application to convert your data to SPSS format.

1
Open the application

Launch via the live URL or your local RStudio instance.

2
Navigate to the Data Converter tab

This is the main conversion workspace.

3
Upload your data file

Click Choose Data File and select any supported input format.

4
Review the preview panels

Data Preview — first rows of the dataset. Data Structure — variable types and unique values. Variable Analysis — how each variable will be processed.

5
Click "Convert to SPSS format"

The app processes and transforms your data automatically.

6
Review the SPSS Metadata panel

Confirm variable names, labels, and value labels look correct.

7
Download your SPSS file

Enter a filename and click Download SPSS File.

Final Steps in SPSS

After downloading and opening the .sav file in SPSS Statistics, one manual step is needed to ensure variables are analyzed correctly.

⚠️ Important

Data2SPSS automatically sets categorical variables' measure to scale. You must adjust the measure type manually to nominal or ordinal in SPSS Variable View — otherwise SPSS may treat categorical variables as continuous numbers, producing incorrect results.

How to Set the Measure Type

Go to Variable View (tab at the bottom of the SPSS window) → find the Measure column → assign the correct type for each categorical variable:

Nominal

Categories with no natural order. Examples: gender, treatment group, country of origin.

Ordinal

Categories with a meaningful order. Examples: education level, Likert scales, pain severity ratings.

How It Works

Variable Name Cleaning

Variable names are automatically cleaned to comply with SPSS naming requirements:

IssueAction Taken
SpacesRemoved entirely
Special charactersEliminated
Names longer than 64 charactersTruncated
Names starting with a numberPrefixed with a letter
Duplicate variable namesMade unique by appending numbers

Value Label Assignment

The application automatically detects categorical variables and generates value labels, preserving the semantic meaning of coded values when the file is opened in SPSS.

Limitations

The following known limitations apply to the current version of Data2SPSS:

📏
Nominal vs. Ordinal Not Distinguished

The tool does not automatically distinguish nominal from ordinal categorical variables. Users must adjust measure types manually in SPSS Variable View after import.

❓
Missing Value Definitions Not Propagated

User-defined missing value codes from source files are not carried over. These should be defined in SPSS after import.

⚡
Performance Depends on System Resources

Very large datasets may require longer processing times depending on available memory and CPU.

🌐
Internet Required for Web Version

The hosted version requires an active internet connection. For offline use, install and run the app locally.
